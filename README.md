# 乡村教育助手（Rural Edu-Agent）

> 面向乡村教师的 AI 教学助手，群聊发一句话即可自动生成教案、写入多维表格、创建文档、同步日历、推送教学简报。

## 🌟 项目简介

乡村教师每天面临繁重的备课负担，却缺乏有效的数字化工具支持。本项目基于 SOLO + 飞书开放平台，打造了一个零门槛的 AI 教学助手，让乡村教师从重复性备课工作中解放出来。

### 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 📝 教案生成 | 一句话生成结构化教案，适配乡村教学条件 | P0 |
| 📊 教学记录 | 自动写入飞书多维表格，记录教学进度 | P0 |
| 📅 教学日历 | 自动创建课程日程，同步到飞书日历 | P1 |
| 📰 教学简报 | 每周自动生成教学周报/简报 | P1 |
| 📦 教学包生成 | 一键打包课件、练习题、教学素材 | P2 |
| 🤖 AI 答疑 | 学生常见问题自动解答 | P2 |

## 🚀 快速开始

### 前置条件

- [SOLO](https://solo.ai) 已安装
- [lark-cli](https://github.com/larksuite/cli) 已安装并完成认证
- 飞书账号已开通相关 API 权限

### 安装

1. 将 `rural-edu-agent/` 文件夹放入 SOLO Skills 目录
2. 重启 SOLO

### 使用

在飞书群聊中发送自然语言即可触发：

```
# 生成教案
帮我生成一份三年级数学《分数的初步认识》教案

# 记录教学
今天上了三年级数学《同分母分数加减法》，2课时

# 生成简报
生成本周教学简报

# 创建日历
每周一三五上午第一节上三年级数学课

# 生成教学包
帮我出一份三年级数学《分数的初步认识》教学包

# AI 答疑
学生总是搞不懂为什么 1/2 和 2/4 一样大
```

## 🔧 首次使用配置

### 1. 创建教学记录表

在飞书中创建多维表格，包含以下字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 教学日期 |
| 学科 | 文本 | 数学/语文/英语等 |
| 年级 | 文本 | 一年级~九年级 |
| 班级 | 文本 | 如"三年级1班" |
| 课题 | 文本 | 教学课题名称 |
| 课时数 | 数字 | 当日课时数 |
| 教学类型 | 文本 | 新课/复习/练习/考试 |
| 教学反思 | 文本 | 教学反思与改进 |
| 教案链接 | 文本 | 关联的教案文档链接 |
| 状态 | 文本 | 已完成/进行中/待上课 |

### 2. 配置参数

将多维表格的 token 和 table-id 记录下来，使用时替换命令中的占位符：
- `{多维表格token}` → 你的多维表格 token
- `{表格ID}` → 你的表格 ID
- `{群聊ID}` → 目标群聊 ID（用于推送简报）

## 💡 设计亮点

### 乡村教学适配

- 实验类课程提供替代方案（用日常物品替代实验器材）
- 多媒体课件提供板书替代方案
- 考虑大班额情况（分组活动调整）
- 考虑学生基础差异（分层教学设计）

### 零门槛交互

- 自然语言触发，无需学习复杂操作
- 意图自动识别，信息缺失时智能追问
- 一句话完成多步操作（生成教案 + 写入表格 + 创建文档）
