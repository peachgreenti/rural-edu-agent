# 飞书文档（lark-doc）操作参考

## 目的

指导如何在乡村教育助手中操作飞书文档，实现教案文档和教学简报的创建。

## 前置条件

执行操作前，先读取 lark-doc Skill 的完整指南：
- 读取 `lark-doc` SKILL.md 了解基础操作
- 读取 `lark-doc-create.md` 了解文档创建
- 读取 `lark-doc-xml.md` 了解 XML 格式（默认格式）

## 创建教案文档

### 基本创建

```bash
# 创建教案文档（XML 格式）
lark-cli docs +create --api-version v2 --content '<title>{课题} — 教学设计</title><p><b>学科</b>：{学科}</p><p><b>年级</b>：{年级}</p><p><b>课时</b>：{课时数} 课时</p><h2>一、教学目标</h2><h3>知识与技能</h3><p>{目标内容}</p><h3>过程与方法</h3><p>{目标内容}</p><h3>情感态度与价值观</h3><p>{目标内容}</p><h2>二、教学重难点</h2><p><b>重点</b>：{重点}</p><p><b>难点</b>：{难点}</p><h2>三、教学过程</h2><h3>环节一：导入新课（5分钟）</h3><p>{内容}</p><h3>环节二：探究新知（15分钟）</h3><p>{内容}</p><h3>环节三：巩固练习（10分钟）</h3><p>{内容}</p><h3>环节四：课堂小结（5分钟）</h3><p>{内容}</p><h3>环节五：布置作业（5分钟）</h3><p>{内容}</p><h2>四、板书设计</h2><p>{板书内容}</p><h2>五、教学反思</h2><p>（课后填写）</p>'
```

### 使用 Markdown 格式创建

```bash
# 创建教案文档（Markdown 格式，适合长文档）
lark-cli docs +create --api-version v2 --doc-format markdown --content '# {课题} — 教学设计

## 一、基本信息
- **学科**：{学科}
- **年级**：{年级}
- **课时**：{课时数} 课时

## 二、教学目标
...'
```

## 创建教学简报文档

```bash
# 创建教学简报
lark-cli docs +create --api-version v2 --doc-format markdown --content '# 📰 教学简报 — 第{N}周

## 📊 本周教学概览
...'
```

## 创建教学包文档

```bash
# 创建教学包
lark-cli docs +create --api-version v2 --doc-format markdown --content '# 📦 教学包 — {课题}

## 一、课堂练习题
...'
```

## 文档命名规范

| 文档类型 | 命名格式 | 示例 |
|---------|---------|------|
| 教案 | `{学科}{年级}{课题}教学设计` | `数学三年级分数的初步认识教学设计` |
| 教学简报 | `教学简报第{N}周({日期})` | `教学简报第12周(0512-0516)` |
| 教学包 | `{学科}{年级}{课题}教学包` | `数学三年级分数的初步认识教学包` |

## 文档权限设置

创建文档后，可根据需要设置权限：
- 默认创建的文档权限为创建者可见
- 如需分享给其他教师，可通过飞书文档的分享功能设置

## 注意事项

- 长文档（教案、教学包）推荐使用 Markdown 格式创建，更易维护
- 短文档（简报摘要）可使用 XML 格式
- 创建文档后，将文档链接回写到多维表格的「教案链接」字段
- 文档内容中的特殊字符需要正确转义（XML 格式下 `<`, `>`, `&` 等）
