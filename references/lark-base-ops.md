# 飞书多维表格（lark-base）操作参考

## 目的

指导如何在乡村教育助手中操作飞书多维表格，实现教学记录的读写和管理。

## 前置条件

执行操作前，先读取 lark-base Skill 的完整指南：
- 读取 `lark-base` SKILL.md 了解基础操作

## 教学记录表操作

### 1. 查找/创建教学记录表

```bash
# 搜索已有的教学记录表
lark-cli drive +search --keyword "教学记录" --type bitable

# 如果不存在，需要提示用户先创建
```

### 2. 写入教学记录

当生成教案或完成教学后，将信息写入多维表格：

```bash
# 写入一条教学记录
lark-cli base +record-create \
  --app-token "{多维表格token}" \
  --table-id "{表格ID}" \
  --fields '{
    "日期": "2026-05-17",
    "学科": "数学",
    "年级": "三年级",
    "班级": "三年级1班",
    "课题": "分数的初步认识",
    "课时数": 1,
    "教学类型": "新课",
    "教学反思": "",
    "教案链接": {"text": "查看教案", "link": "{飞书文档链接}"},
    "状态": "已完成"
  }'
```

### 3. 查询教学记录

```bash
# 按时间范围查询
lark-cli base +record-list \
  --app-token "{多维表格token}" \
  --table-id "{表格ID}" \
  --filter '{
    "conditions": [
      {"field_name": "日期", "operator": "isGreaterThanOrEqualTo", "value": ["2026-05-12"]}
    ]
  }'

# 按学科查询
lark-cli base +record-list \
  --app-token "{多维表格token}" \
  --table-id "{表格ID}" \
  --filter '{
    "conditions": [
      {"field_name": "学科", "operator": "is", "value": ["数学"]}
    ]
  }'
```

### 4. 更新教学记录

```bash
# 更新教学反思
lark-cli base +record-update \
  --app-token "{多维表格token}" \
  --table-id "{表格ID}" \
  --record-id "{记录ID}" \
  --fields '{
    "教学反思": "学生对分数概念理解较好，但比较大小还需加强练习",
    "状态": "已完成"
  }'
```

### 5. 统计教学数据

```bash
# 获取全部记录用于统计
lark-cli base +record-list \
  --app-token "{多维表格token}" \
  --table-id "{表格ID}" \
  --page-size 100
```

从返回数据中统计：
- 总课时数：所有记录的「课时数」字段求和
- 学科分布：按「学科」字段分组计数
- 教学进度：已完成课时 / 计划总课时 × 100%

## 表格 Schema 参考

### 教学记录表

| 字段名 | 字段类型 | 说明 |
|--------|---------|------|
| 日期 | 日期 | 教学日期 |
| 学科 | 单选 | 数学/语文/英语/科学/道德与法治/音乐/美术/体育 |
| 年级 | 单选 | 一年级~九年级 |
| 班级 | 文本 | 如"三年级1班" |
| 课题 | 文本 | 教学课题名称 |
| 课时数 | 数字 | 当日课时数，默认 1 |
| 教学类型 | 单选 | 新课/复习/练习/考试 |
| 教学反思 | 文本 | 教学反思与改进 |
| 教案链接 | 超链接 | 关联的教案文档 |
| 状态 | 单选 | 已完成/进行中/待上课 |

### 教学简报数据表（可选）

| 字段名 | 字段类型 | 说明 |
|--------|---------|------|
| 周次 | 文本 | 如"第12周" |
| 起止日期 | 文本 | 如"5/12-5/16" |
| 完成课时 | 数字 | 本周完成课时数 |
| 新增教案 | 数字 | 本周新增教案数 |
| 学科覆盖 | 文本 | 本周涉及的学科 |
| 教学亮点 | 文本 | 本周教学亮点 |
| 改进方向 | 文本 | 需要改进的地方 |
| 简报链接 | 超链接 | 简报文档链接 |

## 错误处理

- 表格不存在：提示用户先创建多维表格，并提供字段模板
- 权限不足：引导用户检查表格权限设置
- 字段不匹配：提示用户检查表格字段是否与模板一致
