# 飞书日历（lark-calendar）操作参考

## 目的

指导如何在乡村教育助手中操作飞书日历，实现教学日程的自动创建和管理。

## 前置条件

执行操作前，先读取 lark-calendar Skill 的完整指南：
- 读取 `lark-calendar` SKILL.md 了解基础操作和日程创建工作流

## 创建教学日程

### 单次课程日程

```bash
# 创建单次课程日程
lark-cli calendar +create \
  --title "{学科} — {课题}" \
  --start-time "2026-05-19T08:00:00+08:00" \
  --end-time "2026-05-19T08:45:00+08:00" \
  --description "年级：{年级}\n班级：{班级}\n课型：{课型}\n教案链接：{链接}" \
  --color 0
```

### 重复课程日程（每周固定课程）

```bash
# 创建每周重复的课程日程
lark-cli calendar +create \
  --title "{学科}课 — {班级}" \
  --start-time "2026-05-19T08:00:00+08:00" \
  --end-time "2026-05-19T08:45:00+08:00" \
  --rrule "FREQ=WEEKLY;BYDAY=MO;UNTIL=20260731T000000Z" \
  --description "学科：{学科}\n班级：{班级}\n课时：{第几节}" \
  --color 1
```

### RRULE 重复规则说明

| 参数 | 说明 | 示例 |
|------|------|------|
| FREQ | 重复频率 | WEEKLY（每周）、DAILY（每天） |
| BYDAY | 星期几 | MO（周一）、TU（周二）、WE（周三）、TH（周四）、FR（周五） |
| UNTIL | 结束日期 | 20260731T000000Z（到7月31日止） |
| COUNT | 重复次数 | 20（重复20次） |

常用重复规则示例：
- 每周一：`FREQ=WEEKLY;BYDAY=MO`
- 每周一三五：`FREQ=WEEKLY;BYDAY=MO,WE,FR`
- 每周一到周五：`FREQ=WEEKLY;BYDAY=MO,TU,WE,TH,FR`

### 日程颜色编码

| 颜色值 | 含义 | 建议用途 |
|--------|------|---------|
| 0 | 蓝色 | 新课 |
| 1 | 绿色 | 复习课 |
| 2 | 紫色 | 练习课 |
| 3 | 红色 | 考试 |
| 4 | 橙色 | 教研活动 |
| 5 | 青色 | 其他 |

## 查询教学日程

### 查看今日课程

```bash
# 使用快捷方式查看今日日程
lark-cli calendar +agenda
```

### 查看一周课程

```bash
# 查看指定日期范围的日程
lark-cli calendar +search \
  --start-time "2026-05-19T00:00:00+08:00" \
  --end-time "2026-05-23T23:59:59+08:00"
```

## 更新教学日程

```bash
# 更新日程（如修改课题）
lark-cli calendar +update \
  --event-id "{日程ID}" \
  --title "{学科} — {新课题}" \
  --description "更新后的描述"
```

## 删除教学日程

```bash
# 删除单个日程
lark-cli calendar +delete --event-id "{日程ID}"
```

## 批量创建课程表

当用户需要创建整周课程表时，按以下步骤执行：

1. 从用户输入提取课程安排信息（课程名、时间、班级）
2. 为每门课程创建重复日程
3. 汇总创建结果，向用户展示课程表

```
📅 课程表已创建！

| 时间 | 周一 | 周二 | 周三 | 周四 | 周五 |
|------|------|------|------|------|------|
| 第1节 | 语文 | 数学 | 语文 | 数学 | 语文 |
| 第2节 | 数学 | 语文 | 数学 | 语文 | 数学 |
| ... | ... | ... | ... | ... | ... |

共创建 {N} 个重复日程
```

## 注意事项

- 创建日程前确认时间段不与已有日程冲突
- 重复日程设置合理的结束日期（学期末）
- 日程描述中包含教案链接，方便快速查看
- 使用颜色编码区分不同课型，一目了然
