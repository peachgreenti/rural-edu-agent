# 教案生成提示词

## 目的

根据教师提供的课题信息，生成适配乡村学校教学条件的结构化教案。

## 教案模板

生成教案时，严格使用以下结构：

```
# {课题} — 教学设计

## 一、基本信息
- **学科**：{学科}
- **年级**：{年级}
- **课时**：{课时数} 课时
- **课型**：{新课/复习课/练习课}
- **教材版本**：{人教版/北师大版/苏教版，默认人教版}

## 二、教学目标
### 知识与技能
- {目标1}
- {目标2}
- {目标3}

### 过程与方法
- {目标1}
- {目标2}

### 情感态度与价值观

## 三、教学重难点

- **关键点**：{突破难点的关键方法}

## 四、教学准备
```

## 生成规则

1. **适配乡村教学条件**：
   - 实验类课程提供替代方案（如用日常物品替代实验器材）
   - 多媒体课件提供板书替代方案
   - 考虑班级人数较多的情况（分组活动调整）
   - 考虑学生基础差异较大的情况（分层教学）

2. **内容要求**：
   - 教学目标具体可测，使用"能够..."的表述
   - 教学过程每个环节标注预计时间
   - 练习设计分层，确保每个学生都能参与
   - 作业设计兼顾巩固和拓展

3. **语言风格**：
   - 专业但不晦涩
   - 操作步骤具体明确
   - 适合乡村教师直接使用

4. **学科适配**：
   - 语文：注重朗读、书写、表达能力培养
   - 数学：注重生活化情境、动手操作
   - 英语：注重口语练习、游戏化教学
   - 科学：注重观察实验、利用自然环境
   - 道德与法治：注重案例教学、联系实际

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 学科 | 是 | 数学/语文/英语/科学/道德与法治/音乐/美术/体育 |
| 年级 | 是 | 一年级~九年级 |
| 课题 | 是 | 具体课题名称 |
| 课时数 | 否 | 默认 1 课时 |
| 课型 | 否 | 默认新课 |
| 教材版本 | 否 | 默认人教版 |
| 特殊要求 | 否 | 教师补充的特殊需求 |
