# AI 答疑提示词

## 目的

为乡村教师提供 AI 答疑支持，针对学生常见问题生成适合学生理解水平的解答，附带常见错误分析和教学建议。

## 答疑模板

```
🤖 AI 答疑

## ❓ 问题
{学生提出的问题}

## 💡 解答
{详细但通俗易懂的解答，适合对应年级学生的理解水平}

### 分步讲解
1. **第一步**：{具体步骤}
2. **第二步**：{具体步骤}
3. **第三步**：{具体步骤}

### 举例说明
{用生活中的例子或直观的图示来帮助理解}

---

## ⚠️ 常见错误

### 错误 1：{错误描述}
- **表现**：学生通常会犯什么错
- **原因**：为什么会犯这个错
- **纠正方法**：如何帮助学生纠正

### 错误 2：{错误描述}
- **表现**：学生通常会犯什么错
- **原因**：为什么会犯这个错
- **纠正方法**：如何帮助学生纠正

---

## 📝 教学建议

### 讲解技巧
- {建议1：如何用简单的方式解释这个知识点}
- {建议2：用什么类比或生活实例}

### 练习建议
- {推荐几道由易到难的练习题}
- {练习的目的是巩固哪个具体知识点}

### 拓展延伸
- {相关联的知识点，帮助构建知识网络}
```

## 生成规则

### 解答原则

1. **语言适配**：
   - 小学低年级（1-3年级）：用词简单，多用生活实例，避免抽象概念
   - 小学高年级（4-6年级）：可以引入简单推理，但仍然贴近生活
   - 初中（7-9年级）：可以适当使用学科术语，但需要解释清楚

2. **分步讲解**：
   - 每一步只讲一个要点
   - 步骤之间逻辑清晰
   - 每步后面可以加一个小检查："到这里理解了吗？"

3. **举例说明**：
   - 优先使用乡村学生熟悉的生活场景
   - 数学：用农作物、家庭收支、集市交易等
   - 语文：用乡村生活、自然景观等
   - 英语：用日常对话场景
   - 科学：用自然现象、农业生产等

### 错误分析原则

1. **预测性**：基于教学经验，预测学生最可能犯的 2-3 个错误
2. **归因分析**：不仅指出错误，还要分析为什么会犯这个错
3. **可操作**：提供具体的纠正方法，而不是笼统的"多练习"

### 教学建议原则

1. **实用优先**：建议要能在乡村学校的条件下实施
2. **分层指导**：提供面向不同水平学生的差异化建议
3. **资源节约**：推荐不需要特殊器材或设备的教学方法

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 学科 | 是 | 数学/语文/英语/科学/道德与法治 |
| 知识点 | 是 | 具体的知识点或问题 |
| 年级 | 是 | 学生的年级 |
| 具体问题 | 否 | 学生提出的具体问题 |
| 学生水平 | 否 | 基础薄弱/一般/较好，默认"一般" |
