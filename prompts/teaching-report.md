# 教学简报生成提示词

## 目的

基于多维表格中的教学记录数据，自动生成结构化的教学周报/简报。

## 数据来源

从多维表格「教学记录」中读取本周数据，统计以下指标：
- 完成课时数
- 覆盖学科
- 新增教案数
- 教学类型分布（新课/复习/练习/考试）
- 教学反思要点

## 简报模板

```
# 📰 教学简报 — 第{N}周（{起止日期}）

---

## 📊 本周教学概览

| 指标 | 数据 |
|------|------|
| 完成课时 | {N} 节 |
| 覆盖学科 | {学科列表} |
| 新增教案 | {N} 份 |
| 教学天数 | {N} 天 |

## 📚 各学科教学详情

### {学科1}
- **课时数**：{N} 节
- **教学内容**：{课题列表}
- **教学进度**：{当前进度描述}

### {学科2}
- **课时数**：{N} 节
- **教学内容**：{课题列表}
- **教学进度**：{当前进度描述}

## ✨ 本周教学亮点

1. {亮点1：具体的教学成果或突破}
2. {亮点2：学生的积极表现}
3. {亮点3：教学方法上的创新}

## 🔍 教学反思与改进

### 需要关注的问题
1. {问题1：学生掌握薄弱的知识点}
2. {问题2：教学方法需要调整的地方}

### 下周改进计划
1. {改进措施1}
2. {改进措施2}
3. {改进措施3}

## 📅 下周教学计划

| 日期 | 学科 | 课题 | 课型 |
|------|------|------|------|
| {日期} | {学科} | {课题} | {课型} |

## 💡 教学小贴士

{根据本周教学情况，提供1-2条实用的教学建议或资源推荐}

---

> 📋 本简报由乡村教育助手自动生成
> 生成时间：{生成时间}
```

## 生成规则

1. **数据驱动**：简报中的所有数据必须来自多维表格的实际记录，不编造数据
2. **亮点提炼**：从教学反思中提取积极内容，用鼓励性语言表述
3. **问题分析**：客观分析教学中的不足，提供可操作的改进建议
4. **计划衔接**：下周计划应与本周进度自然衔接
5. **语言风格**：简洁专业，适合学校内部汇报使用

## 特殊情况处理

- **本周无教学记录**：生成简报时注明"本周暂无教学记录"，不编造数据
- **数据不完整**：基于已有数据生成，缺失部分标注"暂无数据"
- **教学反思为空**：跳过反思分析部分，或根据教学内容给出通用建议

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 时间范围 | 否 | 默认本周（周一至周日） |
| 多维表格 token | 是 | 教学记录表的 token |
| 表格 ID | 是 | 教学记录表的 table-id |
| 群聊 ID | 否 | 如需推送简报到群聊 |
