# 示例：日常教学记录

## 场景一：记录今日教学

### 用户输入
```
今天上了三年级数学《同分母分数加减法》，2课时，学生掌握得不错
```

### 意图识别
```json
{
  "intent": "教学记录",
  "extracted_info": {
    "subject": "数学",
    "grade": "三年级",
    "topic": "同分母分数加减法",
    "duration": 2,
    "date": "今天",
    "extra": "学生掌握得不错"
  }
}
```

### 助手输出
```
📊 教学记录已更新！

├── 日期：2026-05-17
├── 学科：数学
├── 课题：同分母分数加减法
├── 课时：2 节
├── 班级：三年级1班
└── 备注：学生掌握得不错

📈 本学期累计：86 节课
```

## 场景二：查看教学进度

### 用户输入
```
这学期数学教到哪里了？

### 意图识别
```json
{
  "intent": "教学记录",
  "extracted_info": {
    "subject": "数学",
    "query_type": "进度查询",
    "time_range": "本学期"
  }
}
```

### 助手输出
```
📊 本学期数学教学进度

├── 已完成课时：42 节
├── 教学进度：70%
├── 当前单元：第七单元 分数的初步认识
└── 已完成课题：
    ├── 分数的初步认识 ✅
    ├── 同分母分数比较 ✅
    └── 同分母分数加减法 ✅

📋 下一课：第八单元 小数的初步认识
```

## 场景三：创建教学日历

### 用户输入
```
帮我安排每周一三五上午第一节上三年级数学课

### 意图识别
```json
{
  "intent": "教学日历",
  "extracted_info": {
    "subject": "数学",
    "grade": "三年级",
    "day_of_week": "周一、周三、周五",
    "time_slot": "上午第一节",
    "repeat_rule": "每周"
  }
}
```

### 助手输出
```
📅 教学日历已创建！

├── 课程：数学课 — 三年级1班
├── 时间：每周一/三/五 第1节（08:00-08:45）
├── 重复：每周重复至学期结束
└── 共创建 3 个重复日程

💡 日历中已用 🟢绿色 标记，方便识别
```
