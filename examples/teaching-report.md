# 示例：教学简报生成

## 用户输入

```
生成本周教学简报
```

## 意图识别结果

```json
{
  "intent": "教学简报",
  "extracted_info": {
    "date_range": "本周",
    "extra": null
  }
}
```

## 执行步骤

1. ✅ 意图识别 → 教学简报
2. ✅ 从多维表格读取本周教学记录
3. ✅ 统计教学数据
4. ✅ 生成结构化简报
5. ✅ 创建飞书简报文档
6. ✅ 推送到教学群聊

## 助手输出


## 简报内容摘要

### 本周教学概览
| 指标 | 数据 |
|------|------|
| 完成课时 | 15 节 |
| 覆盖学科 | 语文、数学、英语 |
| 新增教案 | 5 份 |
| 教学天数 | 5 天 |

### 各学科详情
- **语文（5节）**：《荷花》《小虾》课文学习，习作"我的植物朋友"
- **数学（6节）**：分数的初步认识（新课）、同分母分数比较（练习）
- **英语（4节）**：Unit 5 Do you like pears?

### 教学亮点
1. 数学课折纸活动效果好，学生积极参与
2. 语文课《荷花》朗读比赛激发了学生兴趣

### 改进方向
1. 英语课口语练习时间不足，建议增加 pair work 环节
2. 部分学生对分数概念理解不够深入，下周需加强
