# 示例：AI 答疑

## 用户输入

```
学生总是搞不懂为什么 1/2 和 2/4 是一样大的，怎么解释比较好？
```

## 意图识别结果

```json
{
  "intent": "AI答疑",
  "extracted_info": {
    "subject": "数学",
    "grade": "三年级",
    "topic": "分数的相等（等值分数）",
    "extra": "1/2 和 2/4 为什么相等"
  }
}
```

## 助手输出

```
🤖 AI 答疑

❓ 问题：为什么 1/2 和 2/4 是一样大的？

💡 解答：
想象你有一块月饼：
- 1/2 = 把月饼切成 2 份，拿其中 1 份
- 2/4 = 把月饼切成 4 份，拿其中 2 份

你拿到的月饼其实是一样多的！只是切法不同而已。

分步讲解：
1. 拿一张纸，对折一次 → 展开看到 2 份，涂 1 份 → 这就是 1/2
2. 同一张纸，再对折一次 → 展开看到 4 份，涂 2 份 → 这就是 2/4
3. 对比两次涂色部分 → 完全一样大！

⚠️ 常见错误：
• 错误：学生认为"4 比 2 大，所以 2/4 > 1/2"
  原因：只看了分母，忽略了分子
  纠正：用折纸操作让学生亲眼看到

📝 教学建议：
• 用折纸活动让学生亲手操作，比纯讲解有效得多
• 可以用"分披萨"的情境：2 人分 vs 4 人分
```
